# Images Folder

Place your images here:

- `hero-background.jpg` - Desktop hero section background (espresso brewing image)
- `hero-background-mobile.jpg` - Mobile hero section background
- `logo.png` - Main logo (circular with ice cream cone/construction hat)
- `home-icon.svg` - Home icon (optional, currently using SVG inline)

## Image Requirements

- **Hero Background (Desktop)**: Recommended size: 1920x1080px or larger
- **Hero Background (Mobile)**: Recommended size: 768x1024px or larger
- **Logo**: Recommended size: 400x400px (will be scaled down)
